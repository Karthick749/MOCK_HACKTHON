# Customer Data Integration & Sales Analytics System

## Project Overview
This project was developed using Informatica Intelligent Cloud Services (IICS) and Power BI.

The system integrates customer, product, address, and transaction data from multiple sources including:
- Oracle Database
- CSV Files
- REST APIs
- JSON Files

---

## Features Implemented

### Informatica IICS
- Customer Data Cleaning & Standardization
- Product Data Integration
- Address Data Integration
- JSON Parsing using Hierarchy Parser
- Lookup Transformation
- Aggregator Transformation
- Router Transformation
- Sequence Generator
- Incremental Loading Concept
- Reject Handling
- Email Notification
- Linear Taskflow

---

## Database Features
- Primary Keys
- Foreign Keys
- Constraints
- Normalized Tables

---

## Power BI Dashboard
Dashboard includes:
- KPI Cards
